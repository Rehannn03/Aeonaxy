## Installation

Install my-project with npm

```bash
  git clone
  npm init
  npm run dev
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PORT`
`MONGO_URL`
`DB_NAME`
`JWT_SECRET`
`JWT_EXPIRY`
`JWT_REFRESH_SECRET`
`JWT_REFRESH_EXPIRY`
`CLOUDINARY_NAME`
`CLOUDINARY_API_KEY`
`CLOUDINARY_API_SECRET`
